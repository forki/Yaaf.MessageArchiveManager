TypeDescription
IExtendedUserArchivingStore Interface to manage your Chat-Archive. This interface should allow anybody to implement http://xmpp.org/extensions/xep-0136.html or http://xmpp.org/extensions/xep-0313.html on top It is also possible to use another server with http://xmpp.org/extensions/xep-0136.html as backend! But also multiple frontends are possible (Webserver for example).
